  Usage:
      $0 COMMAND [options]

  Commands:

      help
                          print usage information

      version
                          show version info and exit

      init
                          initialize a new repo

      addfile <file> [file2] [file3] ...
                          makes the listed files available to commit

      commit
                          make new commit with new and/or changed whitelisted files

      list
                          list tracked files (don't necessariy exist)

      git <git command>
                          pass command through to git (within ssdm context)
